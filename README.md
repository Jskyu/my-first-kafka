## My First Kafka
> Spring Kafka 공부 레파지토리

### Tech
- `Java 11`
- `Spring Boot 2.7.9`
- `Apache Kafka 2.13-2.8.0`
- `React`

__built with__
- `IntelliJ`

### How to Start
```
1. Mapping Kafka configuration ({projectDir}/src/main/resource/application.yml)
2. gradlew compileJava
3. gradlew bootRun
4. npm start --prefix src\front/
```
